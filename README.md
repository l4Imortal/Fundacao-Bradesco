# 📚 Projetos - Cursos da Fundação Bradesco

Este repositório contém projetos e exercícios desenvolvidos durante os cursos realizados na **Fundação Bradesco (Escola Virtual)**.

O objetivo é registrar meu aprendizado em programação e desenvolvimento web, além de servir como portfólio para demonstrar minhas habilidades e evolução na área de tecnologia.

## 📂 Estrutura do Repositório

Atualmente o repositório possui as seguintes pastas:

📁 **python**
Projetos e exercícios desenvolvidos durante cursos de Python, incluindo práticas de lógica de programação e conceitos básicos da linguagem.

📁 **site-simples**
Projeto de um site simples desenvolvido com tecnologias básicas da web para praticar estrutura de páginas e estilização.

📁 **certificados**
Pasta onde estão armazenadas as imagens ou PDFs dos certificados de conclusão dos cursos.

## 🚀 Tecnologias Utilizadas

* Python
* HTML
* CSS

## 🎓 Certificados

Os certificados obtidos durante os cursos estão disponíveis na pasta **certificados**, comprovando a conclusão das formações.
